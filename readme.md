# **Llefià Music Fest**

### **Descripción breve de la web**
Este sitio web está hecho para informar a la gente sobre un festival de música local en Llefià llamado ***Llefiá Music Fest***. Es una web que utiliza colores oscuros y algunos bordes brillantes para hacer contraste. 

### **Estructura de los ficheros**
La estructura es bastante básica, los archivos principales como el **index**, el **styles** y el **read me** están en el primer nivel. En el segundo nivel tenemos dos carpetas, una que se llama images que contiene la imagen del escenario de la página de **"cartel.html"**. Luego, en la siguiente carpeta llamada paginas tenemos las **páginas detalle** de los diferentes elementos del menú de navegación en la parte superior, que són: **cartel, contacto y horario**.

### **Instrucciones de uso de la web**
Para visualizar de forma correcta esta web, deberás abrir desde el archivo **"index.html"** y tener todos los demás archivos y carpetas en su respectivo nivel, sino no funcionará de forma correcta.
